# C# DDD Bootstrap (base / project skeleton)

## Starting the server
* Local using:
    * `dotnet run --project apps/Mooc/Backend/Backend.csproj`
* Docker using:
    * `docker-compose up`
    
And then going to http://localhost:8030/health-check to check all is ok.
